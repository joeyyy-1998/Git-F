  - [x] Js中var,let,const的区别：
  
    1、var声明的变量属于函数作用域，而let和const声明的变量属于块级作用域；（js作用域在上篇文章） 
    2、var声明的变量存在变量提升，而let和const没有
    3、var声明的变量可以重复声明，而在同一块级作用域，let变量不能重新声明，const常量不能修改（对象的属性和方法，数组的内容可以修改）
